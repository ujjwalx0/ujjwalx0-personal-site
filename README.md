# Portfolio

This project presents a comprehensive portfolio highlighting my expertise, professional journey, and key projects. Built with a clean and modern design, the portfolio ensures an intuitive and engaging experience across various devices.

## Getting Started

### Prerequisites

Ensure you have Node.js `16.0.0` or higher and npm `9.0.0` or higher installed.

### Setup

1. Download the project files from [this link](#).
2. Once downloaded, extract the files and open the project in [Visual Studio Code](https://code.visualstudio.com/download).

### Installation

After opening the project in Visual Studio Code, run the following command in the terminal to install all dependencies:

```
npm install
```

### Usage

To start the project locally and view it in your browser:

```
npm run dev
```

This will launch the project on `http://localhost:5173/`.

### Build

To create an optimized production build:

```
npm run build
```

You can preview the production build locally using:

```
npm run preview
```
